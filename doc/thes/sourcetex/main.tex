\begin{abstract}
  Velvet is a Dafny-style verifier for imperative programs in Lean, built on
Loom. This thesis extends Velvet with cost-aware verification, enabling
programmers to specify and verify resource bounds alongside functional
correctness. The extension is implemented via syntax-directed instrumentation
using Lean’s elaboration mechanism, allowing cost reasoning to reuse the
existing verification infrastructure. We demonstrate the approach on some
examples.
\end{abstract}
